pm-wicket-archetype
===================

Template to create a web application with Wicket + Guice + Hibernate

## Create your project

```bash
mvn archetype:generate -DarchetypeGroupId=com.premiumminds -DarchetypeArtifactId=pm-wicket-archetype
```
